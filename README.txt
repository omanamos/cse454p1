Instructions
------------
1. Run make file to build
2. Run the java file
	- java Main [path to training data] [path to test data]
	- if second two parameters aren't provided, the program will assume the folders 'classifier_data_small_train' and 'classifier_data_test' are in the current directory
3. The program will output to test.output in the current directory.
